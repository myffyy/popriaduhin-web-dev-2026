# Лабораторная работа № А-2: Табулирование функций

Этот документ содержит полное описание выполненной лабораторной работы, включая структуру проекта, разбор ключевых блоков кода и ответы на контрольные вопросы для подготовки к защите.

## Описание проекта

Целью работы является создание PHP-скрипта, который выполняет табулирование (расчет и вывод в виде таблицы или списка) значений кусочно-заданной функции в определенном диапазоне.

**Ключевые возможности:**

- Расчет значений функции для варианта №10.
- Динамическое изменение типа верстки (5 различных видов) через GET-параметр в URL.
- Обработка деления на ноль.
- Остановка вычислений, если значение функции выходит за заданные пределы.
- Расчет и вывод агрегированных данных: сумма, минимальное, максимальное и среднее значение функции.
- Адаптивный и структурированный дизайн с использованием HTML5 и CSS3.

## Структура файлов

- **`index.php`**: Главный файл, содержащий HTML-разметку страницы и всю логику на PHP.
- **`styles.css`**: Внешний файл стилей, отвечающий за все визуальное оформление страницы.
- **`images/`**: Директория для хранения изображений, в данном случае — логотипа университета.
- **`README.md`**: Данный файл с описанием проекта.

## Описание ключевых блоков кода

### `index.php`

#### HTML-структура и CSS

- **Семантическая разметка**: Страница построена с использованием семантических тегов HTML5: `<header>`, `<main>`, и `<footer>`, что улучшает структуру и доступность.
- **"Липкий" подвал (Sticky Footer)**: Реализован с помощью CSS Flexbox. Контейнеру `<body>` заданы `display: flex` и `flex-direction: column`, а основному контенту `<main>` — `flex: 1 0 auto`. Это заставляет `<main>` занимать все доступное пространство, прижимая подвал к низу страницы.

#### Инициализация PHP-переменных

В начале PHP-скрипта инициализируются все управляющие параметры:

```php
$type = strtoupper($_GET['type'] ?? 'A'); // Тип верстки из URL, по умолчанию 'A'
$start_value = -16;                      // Начальное значение аргумента X
$encounting = 17;                        // Количество вычисляемых значений
$step = 2;                               // Шаг изменения аргумента
$min_value = -100;                       // Минимальная граница значения функции
$max_value = 100;                        // Максимальная граница значения функции
$layout_names = [ /* массив с названиями версток */ ];
```

#### Основной цикл вычислений

Центральная часть логики. Используется цикл `for`, который соответствует структуре из листингов методички.

```php
for ($i = 0, $x = $start_value; $i < $encounting; $i++, $x += $step) {
    // ... логика ...
}
```

- Цикл выполняется `$encounting` раз.
- Переменная `$i` служит счетчиком итераций.
- Переменная `$x` (аргумент функции) инициализируется стартовым значением и увеличивается на `$step` в каждой итерации.
- Выполнение может быть досрочно прервано оператором `break`, если значение функции выйдет за рамки `[$min_value, $max_value]`.

#### Вычисление функции (Вариант 10)

Внутри цикла находится блок `if/elseif/else`, который определяет, по какой из трех формул будет вычисляться значение функции `f(x)` в зависимости от текущего значения `$x`.

```php
if ($x <= 10) {
    if ($x == 0) {
        $f = 'error'; // Обработка деления на ноль
    } else {
        $f = 3 / $x + $x / 3 - 5;
    }
} elseif ($x < 20) {
    $f = ($x - 7) * ($x / 8);
} else { // x >= 20
    $f = 3 * $x + 2;
}
```

#### Блок вывода результатов

После всех вычислений результаты выводятся на страницу. Конструкция `switch ($type)` используется для выбора одного из пяти способов отображения данных в зависимости от выбранного типа верстки.

```php
switch ($type) {
    case 'B': // Маркированный список <ul>
        // ...
        break;
    case 'D': // Таблица <table>
        // ...
        break;
    // ... и т.д. для всех 5 типов
}
```

## Ответы на контрольные вопросы

**1. Что такое цикл?**

- Цикл — это управляющая конструкция в программировании, которая позволяет многократно выполнять блок кода до тех пор, пока заданное условие истинно (или ложно).

**2. Что такое условный оператор?**

- Условный оператор (например, `if-else`) — это конструкция, которая выполняет различные блоки кода в зависимости от истинности или ложности заданного логического условия.

**3. Какие виды циклов бывают?**

- В PHP существуют следующие виды циклов:
  - `for`: цикл со счётчиком.
  - `while`: цикл с предусловием (выполняется, пока условие истинно).
  - `do-while`: цикл с постусловием (выполняется хотя бы один раз, а затем проверяет условие).
  - `foreach`: цикл для перебора элементов массивов и объектов.

**4. Чем цикл с предусловием отличается от цикла с постусловием?**

- Цикл с **предусловием** (`while`) сначала проверяет условие и, если оно ложно, может не выполниться ни разу.
- Цикл с **постусловием** (`do-while`) сначала выполняет тело цикла, а затем проверяет условие. Гарантированно выполняется хотя бы один раз.

**5. Чем отличаются операторы `break` и `continue`?**

- `break`: Полностью и немедленно прекращает выполнение цикла.
- `continue`: Прекращает выполнение только _текущей_ итерации и переходит к началу следующей.

**6. Можно ли решить задачу листингов А-2.4 – 6 четвертым путем? Если да, то каким?**

- Да, можно. В листингах показаны решения с `break` в цикле `for`, с `while` и `do-while`. Четвертым путем, хотя и крайне нерекомендуемым из-за нарушения структурности кода, является использование оператора `goto` для выхода из цикла по метке.

**7. Зачем в листинге А-2.5 проверяется условие первой итерации цикла?**

- В условии `while($i<$encounting && ($f>=$max_value || $f<$min_value || !$i))` часть `!$i` добавлена для того, чтобы цикл гарантированно выполнился в первый раз. На первой итерации `$i` равно 0, `!$i` истинно, и все условие `while` становится истинным. Это "костыль" для имитации поведения цикла `do-while`, так как переменная `$f`, от которой зависит условие, еще не вычислена до первого прохода.

**8. Что такое итерация?**

- Итерация — это однократное выполнение тела (блока кода) цикла.

**9. Какой тип данных определяет условие?**

- Логический или булев тип (`boolean`), который имеет два значения: `true` (истина) или `false` (ложь).

**10. Что такое булева алгебра?**

- Это раздел математической логики, который оперирует логическими переменными, принимающими значения "истина" (`true`) или "ложь" (`false`).

**11. Какие операции булевой алгебры (логические операции) Вы знаете?**

- Основные логические операции:
  - `AND` (Логическое "И", `&&`): истинно, только если оба операнда истинны.
  - `OR` (Логическое "ИЛИ", `||`): истинно, если хотя бы один операнд истинен.
  - `NOT` (Логическое "НЕ", `!`): инвертирует значение операнда.
  - `XOR` (Исключающее "ИЛИ"): истинно, если только один из операндов истинен.

**12. Как в РНР округлить значение до заданной точности?**

- С помощью встроенной функции `round()`, которая принимает вторым необязательным аргументом количество знаков после запятой. Пример: `round(3.14159, 2)` вернет `3.14`.

**13. Как в РНР преобразовать вещественный тип к целому?**

- Это можно сделать несколькими способами:
  - Явное приведение типа: `(int) $number`.
  - Функция `intval($number)`.
  - Функции округления `round()`, `floor()` (в меньшую сторону), `ceil()` (в большую сторону).

**14. Какие виды округления используются в РНР?**

- `round()`: Математическое округление до ближайшего целого (или до заданной точности).
- `floor()`: Округление "в пол" (до ближайшего меньшего целого).
- `ceil()`: Округление "в потолок" (до ближайшего большего целого).

**15. Что такое конструкция выбора?**

- Это управляющая конструкция, которая позволяет выбрать один из нескольких блоков кода для выполнения на основе значения переменной. В PHP это `switch-case`.

**16. Почему в примере А-2.2 и других не надо обрабатывать возможность деления на ноли при х=4?**

- В этих примерах возможность деления на ноль при `x=4` **надо** обрабатывать. В тексте методички указано, что "Рассмотренный скрипт содержит сразу несколько погрешностей". Отсутствие обработки деления на ноль в выражении `7/(x-4)` при `x=4` — это одна из таких намеренных погрешностей, которую студент должен найти и исправить.

**17. Как изменить код А-2.3 так, чтобы выводить символ перевода строки `<br>` не в конце сформированной строки с данными, а в ее начале?**

- Нужно проверять, не является ли текущая итерация первой. Если нет, то сначала выводить `<br>`, а затем уже строку с данными.

```php
// Вместо:
echo 'f('.$x.')='.$f.'<br>';

// Сделать так:
if ($i > 0) { // $i - счетчик итераций
    echo '<br>';
}
echo 'f('.$x.')='.$f;
```
