## Описание работы программы

Программа представляет собой один PHP-файл (`index.php`), который динамически формирует страницу с таблицей умножения. Поведение страницы определяется двумя GET-параметрами:

| Параметр | Значения | Назначение |
|----------|----------|------------|
| `html_type` | `TABLE`, `DIV` | Тип верстки таблицы умножения |
| `content` | `2`–`9` | Столбец таблицы умножения (если не задан — выводится вся таблица) |

### Блоки программы

#### 1. Главное меню (`#main_menu`) — шапка страницы
Два пункта-ссылки: **«Табличная верстка»** и **«Блочная верстка»**.
- Передают параметр `html_type` (значения `TABLE` / `DIV`).
- При первой загрузке (параметр не задан) ни один пункт не выделен, по умолчанию используется табличная верстка.
- При выборе пункт выделяется CSS-классом `selected`.
- Ссылки сохраняют текущее значение параметра `content` (сопряжение меню).

#### 2. Основное меню (`#product_menu`) — боковая панель
Пункты: **«Вся таблица умножения»**, **«Таблица умножения на 2»** ... **«Таблица умножения на 9»**.
- Передают параметр `content`.
- По умолчанию (при первой загрузке) выделен пункт «Вся таблица умножения».
- Ссылки сохраняют текущее значение параметра `html_type` (сопряжение меню).

#### 3. Таблица умножения (`#multiplication_table`) — основная область
Зависит от обоих параметров:
- **`html_type`** определяет тип верстки (таблица `<table>` или блоки `<div>`).
- **`content`** определяет содержание (вся таблица или один столбец).

Реализована через четыре PHP-функции:

| Функция | Описание |
|---------|----------|
| `outNumAsLink($x)` | Возвращает число как ссылку `<a>` на соответствующую таблицу (если число ≤ 9), иначе — просто число |
| `outRow($n)` | Выводит столбец таблицы умножения на `$n` (строки вида `NxI=N*I`) |
| `outTableForm()` | Выводит таблицу умножения в табличной форме (`<table>`) |
| `outDivForm()` | Выводит таблицу умножения в блочной форме (`<div>`) с классами `ttRow` / `ttSingleRow` |

#### 4. Информация о таблице (`#footer`) — подвал
Выводит:
- тип верстки (табличная / блочная);
- название таблицы (полная или конкретный столбец);
- текущую дату и время (`date('d.Y.M h:i:s')`).

### Сопряжение меню
Каждая ссылка главного меню сохраняет параметр `content`, а каждая ссылка основного меню сохраняет параметр `html_type`. Это гарантирует, что при переключении одного меню выделение другого не сбрасывается.

## Контрольные вопросы

### 1. Можно ли в ссылке использовать три параметра?

Да. В URL можно передавать произвольное количество GET-параметров, разделяя их символом `&`. Например: `?param1=a&param2=b&param3=c`.

### 2. Можно ли передавать в ссылку параметр без значения?
Да. Например, `?myparam` — параметр `myparam` будет присутствовать в массиве `$_GET` с пустой строкой в качестве значения. Функция `array_key_exists('myparam', $_GET)` вернёт `true`, а `$_GET['myparam']` будет равен `""`.

### 3. Какое начальное значение может быть у цикла со счетчиком?
Любое целое (или дробное) число. Начальное значение задаётся в первом выражении оператора `for`. Например, `for($i=0; ...)`, `for($i=-10; ...)`, `for($i=2; ...)` — все варианты допустимы.

### 4. Что такое URL?

**URL** (Uniform Resource Locator) — унифицированный указатель ресурса. Это адрес, по которому можно найти ресурс в сети. Формат: `протокол://хост:порт/путь?параметры#якорь`. Например: `http://example.com/index.php?content=5`.

### 5. Что такое URI?

**URI** (Uniform Resource Identifier) — унифицированный идентификатор ресурса. Более общее понятие, чем URL. Каждый URL является URI, но не каждый URI является URL. URI может быть просто именем ресурса (URN) без указания его местоположения.

### 6. Что такое конкатенация?
Конкатенация — это операция соединения (склеивания) строк. В PHP выполняется оператором `.` (точка). Например: `'Привет, ' . 'мир!'` даст строку `'Привет, мир!'`. Оператор `.=` добавляет строку к существующей переменной: `$s .= 'текст'`.

### 7. Можно ли применить операцию конкатенации к строке и функции?
Да, если функция **возвращает** значение (через `return`). Результат вызова функции будет преобразован в строку и склеен с другой строкой. Например: `echo 'Результат: ' . myFunc();`. Если функция ничего не возвращает (только выводит через `echo`), конкатенация с ней даст пустую строку, а вывод функции произойдёт отдельно, нарушив порядок.
